HTML이란무엇인가

HTML(Hyper Text Markup Languae)은 가장 단순한 형태의 웹 언어이다.

웹 서버에 HTML 문서를 저장하고 있다가 클라이언가 특정 HTML 페이지를 요청하면 해당 HTML 문서를 클라이언트로 전송한다.

그러면 클라이언트는 이 웹 페이지를 해석하여 웹 브라우저에 표현해 주는데 이런 웹 페이지를 정적인(Static) 웹 페이지라고 한다.


HTML참조사이트

TAG : 설명하다

And the twenty-five elements used on the most pages are:

  1. head : 본문을 설명하는 태그
    98.1%

  2. body : 본문
    97.9%

  3. html : 그 body와 head를 감싸는 태그
    97.9%

  4. title : 웹페이지의 제목을 지정
    97%

  5. meta : 웹브라우저가 웹페이지를 해석하는 방식지정
    93.9%
    title과 meta는 본문이 아니라 본문을 설명하는 내용의 태그이다.
    html은 본문과 본문을 설명하는 태그를 구분함.

  6. div : 영역을 설정, 줄 바꿈 기능 가능, 사각형박스로 구역지정
    89.9%

  7. a : anchor(닻),link삽입
    89.6%
    target="_blank"는 링크를 클릭했을 때 새창에서 페이지가 열리게 하는 속성입니다.
    title은 이 링크가 어떤 내용을 담고 있는지를 툴팁으로 보여주는 기능입니다.

  8. script : 스크립트삽입
    88.5%

  9. link : link요소는 다른 리소스와의 연결을 위해 사용
    86.5%
    - href : 링크의 목적지를 나타냅니다.
    - media : 어떤 매체에 적용되는지 나타냅니다.
    - rel : 링크된 대상의 관계를 나타냅니다.
    - type : MIME 타입을 나타냅니다.
    ☞ 설명 :
    - 주로 외부스타일시트나 대체문서 또는 작성자 지정하기위해 사용됩니다.
    - href, rel 속성은 반드시 존재해야 합니다.

  10. img : 이미지를 웹페이지에 추가할때 쓰는 태그
    86.3%

  11. p : paragraph tag, 단락을 나눔, 줄바꿈보다 정보로서 더 가치있음
    81.5%

  12. span : 영역을 설정, 줄바꿈 기능 불가, 문장단위로 구역지정
    75.6%

  13. li :목차,list
    73.8%

  14. ul :list태그의 모태그 list를 구분함
    ul:unordered list(앞에 동그란 점), ol:ordered list(순차적인 리스트,숫자를 자동으로 붙여줌)
    73.7%

  15. br : 줄바꿈, 닫기 기능 필요없음
    70.3%

  16. style : style 요소는 스타일 정보를 문서에 포함
    - media : 스타일이 적용될 매체를 나타냅니다.
    - type : 스타일 언어를 나타냅니다.
    - scoped : 불리언 속성으로, 이것이 설정되었다면 문서 전체가 아니라 스타일 요소의 부모요소에서 나온 서브트리에만 적용됩니다.
    ☞ 설명 :
    - media속성이 생략되었다면 기본값은 "all"입니다.
    - type속성이 존재하지 않는다면, 기본값은 "text/css"입니다.

    60.4%

  17. h1 : h1 ~ h6요소는 각 섹션의 제목
    55.8%

  18. h2 : h1 ~ h6요소는 각 섹션의 제목
    52.7%

  19. input :여기참조
    48.4%

  20. form :여기참조
    46.9%

  21. strong : 진하게
    44.3%

  22. h3 : h1 ~ h6요소는 각 섹션의 제목
    43.1%

  23. table :table요소는 1차원 이상의 데이터를 표의 형태로 나타냅니다.
    - media : 스타일이 적용될 매체를 나타냅니다.
    - type : 스타일 언어를 나타냅니다.
    - scoped : 불리언 속성으로, 이것이 설정되었다면 문서 전체가 아니라 스타일 요소의 부모요소에서 나온 서브트리에만 적용됩니다.
    - 테이블 요소를 레이아웃을 목적으로 사용해서는 안됩니다.
    - 테이블은 다음 요소를 포함하고 있습니다.
    ◦tr요소 : row (횡렬의 셀)로 정의
    - th요소 : column (종렬의 셀)으로 제목셀을 정의
    - td요소 : cell (표안의 항목)로 데이터셀을 정의
    - caption요소 : 표의 제목을 정의
    - thead요소 : 헤더행을 그룹화하여 정의
    - tfoot요소 : 푸터행을 그룹화하여 정의
    - tbody요소 : 본체행을 그룹화하여 정의
    - col요소 : 열의 속성값과 스타일을 위한 그룹화
    - colgroup요소 : 열의 구조적 그룹화. col요소를 포함할 수 있다.

    30.9%

  24. tr : table참조
    30.3%

  25. td : table참조
    30.2%
